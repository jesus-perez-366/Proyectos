## Explicacion
   El verdugo o tambien conocido como el ahorcado es un juego que pone a trabajar tu capacidad deductiva, ya que la finalidad del juego es adivinar la palabra antes de que seas ahorcado.

## Reglas
   Debe adivinar la palabra antes de alcanzar el numero de equivocaciones.

## Instrucciones
1. El juego se incia y solicita un nombre.
        
    ### Ejemplo:

    <img src=imagenes/1.jpg width="450">

2. El jugador ingresa lo solicitado
3. El juego da la Bienvenida, e inicia el juego montrando la palabra a adivinar
      de la siguente manera:

    ### Ejemplo:

    <img src=imagenes/2.jpg width="450">
   
   `Nota: "las lineas representan cada letra de la palabra"`


 4. El juego solicitara que ingrese una letra del alfabeto, es importante que debe ser una y unicamnete una letra del alfabeto, en caso de que ingrese algo distinto el juego le indicara que lo que a ingresado no es correcto y solicitara que nuevamente ingrese una letra.

    ### Ejemplos:
    
    El usuario coloco: 22

    <img src=imagenes/4.jpg width="450">
   

    El usuario coloco: q,

    <img src=imagenes/5.jpg width="450">


    El usuario coloco: %

    <img src=imagenes/6.jpg width="450">
    

    Para cada caso el juego le indicara el error y hasta no ingresar una letra validad no continuara.

    `NOTA: es indiferente para el juego si usa mayuscula o minuscula.`
    
5. Si la letra es correcta y esta contenida en la palabra, el juego le    mostrara la posicion donde esta ubicada y las que faltan. A partir de este momento empzad a activar sus capacidades deductivas ""sherck holmes""

    ### Ejemplo
    
    El usuario coloco: M
    
    <img src=imagenes/8.jpg width="450">

    
    El usuario coloco: a
    
    <img src=imagenes/7.jpg width="450">

    Ahora si la letra usada no se encuentra dentro de la palabra, el juego le indicara que la letra no esta contenida en la palabra y empezara a dibujar al muñeco

    ### Ejemplo

    El usuario coloco: w

    <img src=imagenes/9.jpg width="450">

6. Los pasos 4 y 5 se ejecutaran y obtendra una repuesta del juego en      funcion de lo que usted ingrese.
    
7. Ganador o Perdedor

    7.1 El juego le indicara que usted es el ganado si logra adivinar la palabra antes de equivocarse 8 veces.

    <img src=imagenes/10.jpg width="450">
    

    7.2 El juego le indicara que ha perdido una vez que se haya equivocado 8 veces, ademas el dibujo del muñeco ahorcado estara completado.

    <img src=imagenes/11.jpg width="450">
    

    `Nota: las equivocaciones son aquellas que coloca la letra y esta no esta contenida en la palabra, pero si usted ingresa una letra repetida o un valor distinto a una letra eso no es contabilizado como una equivocacion.`
